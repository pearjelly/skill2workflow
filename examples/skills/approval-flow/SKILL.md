# Approval Flow Skill

Use this skill when a draft must be reviewed and approved before publication.

<HARD-GATE>
Do NOT publish until the user approves the draft.
</HARD-GATE>

## Checklist

1. Explore project context
2. Draft workflow
3. Ask user for approval
4. Publish workflow

## Verification

- Verify the published workflow has an audit trail.
- Check that all required approval events were recorded.
